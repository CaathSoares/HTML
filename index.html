Atributos das tags

  1. Type – “text” /> = isso seria um a tributo para tags.

  2. Id = Ele é um identificador.

  3. Style = Ele é usado para definir um estilo de formatação, como cores, tamanhos de fonte e layout.

  4. Class = Utilizado para aplicar estilos específicos a elementos ou grupos de elementos. Ele ajuda a separar o conteúdo (HTML) da apresentação (CSS) facilitando a estilo de páginas da web.

  5. Number = ele é um campo para só digitação de números.

  6. Text = campo só para texto.

  7. Color = cria um campo para selecionar as cores que desejar.

  8. Src = onde você pode colocar um endereço de imagem para exibir para o usuário.

  9. Width = Seria o tamanho da imagem que você quer que apareça para o usuário.

  10. U = para deixar o texto underline.

  11. Mark = Fazer marcação do texto.

  12. Sup = Para aparecer um número ou uma letra em cima, exemplo: 2,a.

  13. I = deixar o texto em itálico.

  14. Strong = para deixar o texto em negrito.

  15. Small = é usada para definir texto com uma fonte menor ou com uma aparência diminuída em relação ao texto circundante.

  16. Del = em HTML, que é usada para representar texto que foi deletado ou removido.

Não está todos os atributos, que existe no HTML

Lista Ordenadas e Não Ordenadas

Lista Ordenada (ol):Uma lista ordenada é uma lista na qual os itens são organizados em uma ordem específica, geralmente numerada ou com marcadores. Para criar uma lista ordenada, você usa a tag (ol) e, para cada item da lista, utiliza a tag (li) (item de lista). A ordem dos itens na lista é determinada automaticamente pelo navegador.

Lista Não Ordenada (ul):Uma lista não ordenada é uma lista na qual os itens não seguem uma ordem específica, e geralmente são marcados com marcadores. Para criar uma lista não ordenada, você usa a tag (ul) e, para cada item da lista, utiliza a tag (li) (item de lista). A ordem dos itens na lista não tem importância e não é determinada automaticamente pelo navegador.

Texto

H1= em HTML é usada para definir um cabeçalho de nível 1 em uma página da web. Este é o cabeçalho mais importante e geralmente é usado para o título principal ou o cabeçalho mais proeminente em uma página.

H2= em HTML é usada para definir um cabeçalho de nível 2 em uma página da web. Este cabeçalho é ligeiramente menos importante que o (h1) e geralmente é usado para subtítulos ou seções secundárias em uma página.

P= é usada para definir um parágrafo de texto. Ela é uma das tags mais básicas e comuns em HTML e é utilizada para separar blocos de texto em parágrafos distintos.

Blockquote= em HTML é usada para citar texto de outra fonte, muitas vezes para destacar uma passagem de texto longa, como uma citação ou trecho de um artigo, livro ou site externo. Ela geralmente é exibida com um recuo da margem esquerda e, às vezes, é apresentada em itálico.
